Základní HTML tagy a jejich atributy

Tato příručka obsahuje přehled základních HTML tagů a jejich atributů, včetně semantických tagů a atributů pro vstupní pole (input).

1. Základní HTML tagy

<p> - Odstavec

Tag <p> se používá pro vytváření odstavců. Tento tag má sémantický význam a je využíván pro formátování textu.

Toto je příklad odstavce.

<a> - Odkaz

Tag <a> vytváří hypertextový odkaz. Atribut href specifikuje URL, na kterou odkazuje.

Klikni sem pro návštěvu stránky

<button> - Tlačítko

Tag <button> slouží k vytvoření klikatelného tlačítka. Tlačítko může být použito pro odesílání formulářů, spouštění JavaScriptu nebo jakoukoli jinou interakci s uživatelem.

2. Tagy <div> a <span>

<div> - Oddíl

Tag <div> je blokový element, který se používá k seskupení dalších HTML elementů do sekce nebo oddílu. Často se používá pro layouty.

Toto je oddíl vytvořený pomocí tagu <div>.

<span> - Inline element

Tag <span> je inline element a používá se k seskupení textu nebo jiných inline elementů, obvykle za účelem stylování. Na rozdíl od <div> nezasahuje do layoutu.

Tento text je červený.

3. Atributy inputů

<input> - Vstupní pole

Tag <input> slouží k vytváření různých druhů vstupních polí. Níže jsou všechny typy vstupních polí, které HTML podporuje.

Textový vstup

type="text" vytváří textové pole.

Checkbox

type="checkbox" vytváří zaškrtávací políčko.

Radio Button

type="radio" vytváří tlačítko pro výběr jedné možnosti ze skupiny.

Výběr barvy

type="color" umožňuje výběr barvy.

Heslo

type="password" vytváří pole pro zadání hesla, kde se znaky skryjí.

Výběr data

type="date" vytváří pole pro výběr data.

Email

type="email" vytváří pole pro zadání emailové adresy.

Výběr souboru

type="file" umožňuje výběr souboru z počítače.

Rozsah (slider)

type="range" vytváří posuvník pro výběr číselné hodnoty.

Telefonní číslo

type="tel" vytváří pole pro zadání telefonního čísla.

URL

type="url" vytváří pole pro zadání URL adresy.

4. Semantické HTML tagy

<article>

Tag <article> představuje samostatný a nezávislý obsah, jako například článek nebo blogový příspěvek.

Název článku

Tento článek je o nejnovějších trendech v technologiích...

<aside>

Tag <aside> představuje obsah vedlejšího významu, který doplňuje hlavní obsah stránky.

Vedlejší informace nebo reklama.

<details> a <summary>

Tag <details> umožňuje vytvořit rozbalovací obsah. Tag <summary> určuje nadpis rozbalitelného obsahu.

Klikněte sem pro více informací

Zde je skrytý obsah, který se zobrazí po kliknutí.

<figure> a <figcaption>

Tag <figure> představuje samostatný obsah, například obrázek. Tag <figcaption> přidává popis tohoto obsahu.

<footer>

Tag <footer> představuje patičku stránky nebo sekce.

© 2024 Moje stránka. Všechna práva vyhrazena.

<header>

Tag <header> představuje hlavičku stránky nebo sekce, obvykle obsahuje nadpisy a navigaci.

  • Kontakt
