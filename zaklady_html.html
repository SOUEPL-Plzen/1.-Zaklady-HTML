Základní principy HTML5

HTML5 je nejnovější verze jazyka pro strukturování webových stránek. HTML označuje základní prvky, jako jsou nadpisy, odstavce, odkazy a další.

Párové a nepárové tagy

V HTML existují dva druhy tagů: párové a nepárové (samozavírací).

Část <head>

Část <head> obsahuje informace pro prohlížeč, které se obvykle nezobrazují na stránce. Tato část může obsahovat:

  • <title>: Název stránky, který se zobrazuje v záložce prohlížeče.
  • <meta>: Meta tagy slouží k poskytování informací o stránce, jako je kódování znaků, popis nebo klíčová slova.
  • <link>: Pro připojení externích zdrojů, například souborů CSS nebo ikon.
  • <style>: Místní CSS, které ovlivňuje vzhled stránky.
  • <script>: Místní JavaScript, který může ovlivnit chování stránky.

Závěr

HTML5 poskytuje bohatou sadu nástrojů pro tvorbu webových stránek, včetně nových tagů, jako jsou sémantické tagy, vylepšené formuláře a lepší podpora multimédií.
